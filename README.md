This React application allows users to search for real-time weather data for any city in the world.
The app retrieves current weather and a 5-day forecast using the OpenWeatherMap API and displays useful details such as temperature, weather conditions, wind speed, humidity, and icons.

## Features
- Search weather by city
- Displays temperature, weather condition, and icon
- Responsive UI

## Tech Stack
- React
- OpenWeatherMap API
- CSS

## API Used
**OpenWeatherMap Current Weather API**
https://openweathermap.org/current
